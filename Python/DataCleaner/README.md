# 📊 Data Cleaner Pro

Script de automação ETL (Extract, Transform, Load) para limpeza e padronização de bases de dados.

### 🚀 Tecnologias:
- **Pandas:** Manipulação e análise de dados de alta performance.
- **Openpyxl:** Engine para exportação de relatórios em Excel.

### 🛠️ O que ele faz:
1. Remove registros duplicados automaticamente.
2. Trata valores ausentes (NaN) para evitar erros em dashboards.
3. Exporta os dados prontos para análise em formato .xlsx.
