# 🔍 Intelligent Log Parser

Ferramenta de monitoramento de backend que varre arquivos de log em busca de falhas críticas.

### 🚀 Tecnologias:
- **Regex (Re):** Expressões regulares para busca avançada de padrões de texto.
- **File I/O:** Manipulação eficiente de arquivos de sistema.

### 🛠️ O que ele faz:
- Analisa arquivos de log de servidores em tempo real.
- Identifica palavras-chave como 'ERROR' ou 'FAIL'.
- Gera um sumário de incidentes para a equipe de SRE/Infraestrutura.
